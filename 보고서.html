서비스 개요

사용자가 영화를 등록하고 리뷰를 작성하면, 한국어 키워드 기반 감성 분석을 통해 리뷰의 긍정/부정 여부와 감성 점수를 자동으로 산출하는 웹 서비스입니다. 감성 점수는 별점(1~5점)으로 환산되어 영화별 평균 평점으로 표시됩니다.

감성 분석 모델

분석 방식

한국어 긍정/부정 키워드 사전을 기반으로 리뷰 텍스트를 분석합니다. 외부 API나 대용량 모델 없이 서버 내에서 즉시 처리됩니다.

동작 원리

단계내용
1. 키워드 매칭리뷰 텍스트에서 긍정/부정 키워드 포함 여부 확인
2. 점수 계산긍정 키워드 수 ÷ 전체 매칭 수 = 긍정 점수 (0.0 ~ 1.0)
3. 기본값 처리매칭 없을 경우 0.4 (약한 부정) 반환
4. 레이블 결정점수 ≥ 0.5 → positive / 점수 < 0.5 → negative
5. 별점 환산별점 = 1 + 감성점수 × 4 (1점 ~ 5점 범위)
